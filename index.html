Chaos Game in the Browser

Theory & References

Chaos Game formula:

Pn+1 = Pn + r × (Vi - Pn)

Where:

  • Pn = current point
  • Vi = randomly chosen vertex
  • r = step fraction between 0 and 1

This process iteratively moves the current point closer to a randomly chosen vertex of the polygon.
